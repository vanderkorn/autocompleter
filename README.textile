<p>Программа для решения задачи автодополнения слов<br /></p>

<p>Решение состоит из двух частей:</p> 

<ul>
<li>
Базовое. Консольное приложение.<br />
Проект Vdk.AutoCompleter.TestClient
</li>
<li>
Дополнительная. Клиент-серверный сервис. <br />
В случае использования технологии WCF: проект Vdk.AutoCompleter.Wcf.Server и проект Vdk.AutoCompleter.Wcf.Client. <br />
В случае использования технологии: проект Vdk.AutoCompleter.Thirt.Server и проект Vdk.AutoCompleter.Thirt.Client. 
</li>
</ul>

<p>Для того чтобы скомпилить решение можно выполнить командный файл Vdk.AutoCompleter.Build.cmd. Выходные файлы запишутся в директорию Vdk.AutoCompleter.Release. Также в этой директории находятся командный файлы для удобства тестирования.</p> 

=======================================
<p>This program is to decision the problem of finding autocomplete words<br /></p>

<p>Decision of the problem consists of two parts:</p> 

<ul>
<li>
Basic. Standalone application.<br />
Project Vdk.AutoCompleter.TestClient
</li>
<li>
Additional. Сlient-server service. <br />
In the case of technology WCF: project Vdk.AutoCompleter.Wcf.Server and project Vdk.AutoCompleter.Wcf.Client. <br />
In the case of technology THRIFT: project Vdk.AutoCompleter.Thirt.Server and project Vdk.AutoCompleter.Thirt.Client. 
</li>

</ul>

<p>In order that the solution can be compiled to run the command file Vdk.AutoCompleter.Build.cmd. The output files are written to the directory Vdk.AutoCompleter.Release. Also in this directory are batch files for ease of testing.</p>
